JobApplication（ポートフォリオ）

概要

求人への応募状況を管理するための Web アプリケーションです。  
応募先・応募状況・応募サイトなどを一元管理できるようにしました。

Java（Spring Boot）を用いたバックエンド開発の学習を目的として作成しています。

---

作成背景・目的

転職活動を進める中で、  
「どの企業に・どのサイト経由で・現在どのステータスか」を  
管理するのが煩雑だと感じたため、  
学習を兼ねて応募管理アプリを作成しました。

---

主な機能

- 求人応募情報の登録
- 応募一覧の表示
- 応募ステータスの管理  
  （例：応募中／書類選考中／面接中／結果待ち など）

---

使用技術

- Java
- Spring Boot
- Gradle
- Thymeleaf
- MyBatis
